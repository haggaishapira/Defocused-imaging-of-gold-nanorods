

to analyze a defocused imaging of gold nanorods video:

1. download an exemplary video and metadata file from the google drive link:

https://drive.google.com/drive/folders/1gsynd6PAxUGIGaf0NSyfeqLVjxgLTp7t?usp=drive_link

2. run the "gold rotation.m" app in Matlab (developed in version 2021b)  
3. click "load video" and select the video file (the example is a custom ".rdt" extension format but tif files are possible)
4. click "analyze"
5. analysis results should appear in the graphs.
